# tweetly
Connecting people  

#How to install and run locally ?  
$ git clone https://github.com/iAbhishekPundir/social_media  
$ cd social_media/tweetly  
$ npm install  
$ npm start

# Features  
UserFeed  
Explore
Filtering the posts  
Add, Update & Delete Post  
Add & Delete Comment  
Profile Update  
Follow / Unfollow  
Search User  
Like and Comment on Post  
Authentication  
User Signup  
User Login  
User Logout

# Tech Stack Used  
React JS  
React Context API + useReducer  
React Router  
Vanilla CSS for styling  
MockBee for backend
